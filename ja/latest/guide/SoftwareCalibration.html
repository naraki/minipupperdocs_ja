Mini Pupperのソフトウェアキャリブレーション

1.キャリブレーションセットアップ

ツール

キットに含まれる工具に加え、以下のものが必要です。

  • USBキーボード

  • USBマウス

  • HDMIディスプレイ

  • ディスプレイとRaspberry Piの接続用マイクロHDMIケーブル

ステップ1.1 ラズベリーパイを起動する

  • ラズベリーパイとディスプレイをmicroHDMIケーブルで接続する。

  • USB キーボードとマウスを Raspberry pi に接続する。

  • Mini Pupperのバッテリーボタンを3秒間長押しすると、ラズベリーパイが起動します。

  • 画面にUbuntuが表示されれば成功です。Mini Pupperのディスプレイにも顔が表示されているはずです。

ステップ1.2 ログイン

  • 画面上のubuntuボタンを押してログインする。

  • デフォルトのユーザー名：ubuntu

  • デフォルトのパスワード: mangdang

ステップ1.3 キーボードの設定（オプション）

  • キーボードの種類、例えば日本語キーボードを変更したい場合は、この設定が必要です。左下のメニューボタンを押し、settingsと入力すると設定が表示されます。

  • Region & Languageを選択し、Input Sourcesの＋を押す。表示された画面で、otherの3つの点を選択し、Japaneseと入力して選択する。日本語を入力して選択します。

ステップ1.4 ssh用のポートを開く（オプション）

  • PCからUbuntuにssh接続を試みます。

ssh ubuntu@192.168.x.x -p 22

  • PCからssh接続ができない場合は、次のような設定が必要かもしれない。

sudo ufw allow 22
sudo ufw reload

  • ファイアウォールが実行されていない場合は、次のようにしてください。

sudo ufw enable

2.キャリブレーション

ステップ2.1 脚のキャリブレーション（ROS画像用ではない）

以下のキャリブレーション・ビデオクリップをご参照ください。

  • Mini Pupperのキャリブレーションの際、サーボホーンの位置がずれていると、キャリブレーションがうまくいかず、サーボを壊してしまうことがあります。 以下の簡単なガイドに従ってください（ROSイメージではありません）。

  • Mini Pupperが4本の脚を除いて組み立て済みであることを確認してください。

  • 電源を入れ、デスクトップのキャリブレーションツールアイコンをダブルクリックして起動します。

各サーボのキャリブレーションを行う → 「Update」ボタンを押して設定を保存する → 「×」ボタンを押してキャリブレーションを終了し、保存します。

バーを動かすか、バーの近くのスペース位置をクリックして、各サーボのキャリブレーションを行います。

リセットボタン：前回からやり直したい場合。

リストアボタン工場出荷時の設定に戻します。

  • キャリブレーション GUI ツールを使用して、#1 #4 #7 #10 ヒップサーボを水平位置にキャリブレーションします。

  • その後、下の写真のように4本の脚を正しい位置に手で組み立てる。

  • キャリブレーションGUIツールを使って脚の位置を最適化します。各脚について、すべての脚が45度の角度になるようにバーを動かします。脚の角度は、画面上のスライドバーの位置に連動して変化します。ただし、スライドバーが動かない場合は、手順が間違っています。

  • iPhoneの傾きセンサーアプリ、定規、分度器を使って角度を測ることができる。私も定規と分度器を使った。

  • 脚の正確な傾きを測るのは少し難しいので、方眼紙のようなグリッドに脚を置き、前後左右の位置が同じであることを確認し、最終調整するのがよい。

  • すべての脚が図のように45°の傾きになったら、Updateを押して設定を保存します。× を押してキャリブレーションアプリケーションを閉じます。

ステップ2.2コントローラの接続

  • 次の章をご覧ください。

  • コントローラーで操作できればOK。

Mini Pupper2および2Pro用ソフトウェアキャリブレーション

1.キャリブレーションセットアップ

ツール

初心者の場合、キットに含まれる道具に加えて、以下のものが必要である。

  • USBキーボード

  • HDMIディスプレイ

  • ディスプレイとキャリーボードを接続するマイクロHDMIケーブル

PCからMini Pupperにsshで接続し、ステップ1.4を参照してください。

ステップ1.1スタートアップ

  • キャリーボードとディスプレイをmicroHDMIケーブルで接続する。

  • USBキーボードをキャリーボードに接続する。

  • Mini Pupperのバッテリーボタンを3秒間長押しして起動します。

  • 画面にUbuntuが表示されれば成功です。また、Mini PupperのディスプレイにIPアドレスまたは「IP: no IPv4 address」が表示されるはずです。

ステップ1.2 ログイン

  • 画面上のubuntuボタンを押してログインする。

  • デフォルトのユーザー名：ubuntu

  • デフォルトのパスワード: mangdang

ステップ1.3 WiFi。

  • WiFi設定ファイル(50-cloud-init.yaml)を編集し、WiFiを自分のものに設定する。

sudo vi /etc/netplan/50-cloud-init.yaml
# Edit and save
sudo netplan apply
sudo reboot

  • mini_pupper_web_controllerを参照することもできます。

ステップ1.4 HDMIディスプレイなしでssh用のポートを開く

  • Mini Pupperが使用するデフォルトのWiFiを設定するには、携帯電話を使用します。

    WiFi名: Mangdang

    パスワード: mangdang

  • ロボットの電源を入れ、LCDディスプレイをチェックする

    LCDに "IP: no IPv4 address "と表示されたら、ロボットをもう一度再起動してください。

  • LCDにIPアドレスが表示されたら、PCからロボットにssh接続します。

ssh ubuntu@192.168.x.x

  • PCからssh接続ができない場合は、次のような設定が必要かもしれない。

sudo ufw allow 22
sudo ufw reload

  • ファイアウォールが実行されていない場合は、次のようにしてください。

sudo ufw enable
